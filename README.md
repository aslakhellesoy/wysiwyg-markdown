## Content Editable

See https://developer.mozilla.org/en/docs/Rich-Text_Editing_in_Mozilla

The commands that are relevant to Markdown are:

* bold
* formatBlock (maybe)
* heading
* indent
* insertBrOnReturn
* insertHorizontalRule
* insertHTML (maybe)
* insertImage
* insertOrderedList
* insertUnorderedList
* insertParagraph
* insertText
* italic
* outdent
* strikeThrough (exists in GFM)
